## Features

* 🌐 **Rejoin Server:** -Rejoins the server if anything goes wrong!
* 💻 **Slient-Aim:** -Shoots people without your effort to aim!
* 🚀 **Aimbot:** -Automaticlly Aims at the nearby player!
* ⚙️ **FOV & Resolution:** -Changes how you see.
* 🌐 **Server-Hop** -Server hops if the server hates you.
* 🎮 **Trigger-Bot:** -Auto-Shoots at the player you look at.
* 👤 **ESP:** -Find and locates where people are at
